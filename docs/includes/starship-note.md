???+ note

    **Nerd Fonts:**

    :    The [Starship](https://starship.rs) terminal prompt requires a "Nerd Font" to function properly. The Nerd Fonts project patches open-source, developer targeted *(monospace)* fonts with a large number of glyphs *(icons)* from popular ‘iconic fonts’ such as [Font Awesome](https://github.com/FortAwesome/Font-Awesome), [Devicons](http://vorillaz.github.io/devicons/), [Octicons](https://github.com/github/octicons), and others. The configuration file on this page contains the UTF-8 codes for the symbols, but since this site does not include a compatible Nerd Font they will not be rendered properly.
        
        + You can download a Nerd Font from the official [Nerd Fonts](https://www.nerdfonts.com/) website. I prefer `NotoMono NerdFont` and `JetBrainsMono NerdFont`.
        + Once you install a Nerd Font on your system, you can set the font in your terminal application for the Starship symbols to render properly.
