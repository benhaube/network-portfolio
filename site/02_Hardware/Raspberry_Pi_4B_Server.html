    Raspberry Pi 4B Server

    Raspberry Pi Docs  Debian Docs

    Device Overview

    • Role: Secondary DNS server (Technitium Cluster), CUPS print server, Home Assistant server
    • Hostname(s): pi-server
    • Location:
      • Office
      • Printer-Cart
    • OS / Firmware:   Debian Linux 13 (Trixie)
    • Credentials:
      •  Bitwarden SSH key: "admin@pi-server"

    Maintenance & Notes

    Critical Configurations

    Chrony:

    • Do not modify NTP settings or enable systemd-timesyncd. It has been replaced with chrony.

    Logs:

    • Logs are configured with log2ram to reduce wear on the MicroSD card.

    Email Notifications:

    • This server has email notifications configured for new SSH sessions and unattended-upgrades. See SSH Login Email Notification for documentation.

    Update Process:

    • apt update && apt upgrade (Standard Debian)
    • unattended-upgrades enabled for critical bug and security fixes.

    Backup Policy:

    • Image of MicroSD card stored on ZimaBoard 2 NAS /media/Quick-Storage/Backup directory.
